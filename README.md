# Hospitality Data Transformation Project 📊
## Project Overview
This project demonstrates the transformation of raw, messy Property Management System (PMS) data into a structured, analysis-ready format.

## Key Achievements
* **String Parsing:** Utilized `LEFT`, `RIGHT`, and `FIND` functions to segment guest identities for CRM readiness.
* **Temporal Analysis:** Converted date strings into `Day_of_Week` categories using `TEXT` functions to identify peak check-in patterns.
* **Business Logic:** Implemented conditional `IF` statements to tier revenue ($300+ as "Premium").
* **Data Integrity:** Eliminated duplicates and standardized currency formatting for 100% reporting accuracy.

* **Data Transformation Formulas**
